## 安装方法

### 方式一：DMG 安装包（推荐）

1. 下载 `OmniAi.dmg`
2. 双击打开，把 `OmniAi.app` 拖入应用程序文件夹
3. 在应用程序中双击启动

---

### ⚠️ 提示"OmniAi 已损坏，无法打开"？

这是 macOS 的安全机制，不是真的损坏。两步解决：

**第一步**：打开「终端」（在启动台 → 其他 → 终端）

**第二步**：复制粘贴以下命令，按回车：

```bash
xattr -cr /Applications/OmniAi.app
```

**第三步**：重新双击 OmniAi 即可正常打开。

> 这条命令只是告诉 macOS"我信任这个 App"，安全无副作用。
