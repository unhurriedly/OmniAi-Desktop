## ✨ What is OmniAi?

OmniAi is a Mac app that lets you use all major AI tools in one place:

- ChatGPT
- Claude
- Gemini
- DeepSeek
- and more...

No more switching tabs. No more re-login.

---

## ⚡ Why OmniAi?

- ⚡ Instant switching between AI tools  
- 🧠 Sessions stay alive (no reload, no login again)  
- 🖥 Menu bar app, always one shortcut away  
- 🧩 Clean, minimal interface
